# 🐾 HuellaVet

HuellaVet es una aplicación web para gestionar reservas de servicios veterinarios.

La plataforma permitirá que los clientes registren sus mascotas, consulten los servicios disponibles y agenden citas con un veterinario independiente. El veterinario podrá administrar sus servicios, horarios disponibles y citas desde un panel administrativo.

## Objetivo del proyecto

Facilitar el agendamiento de citas veterinarias, mejorar la organización del profesional y evitar cruces de horarios o pérdida de información.

## Funcionalidades principales

- Registro e inicio de sesión.
- Roles de cliente y veterinario administrador.
- Registro y gestión de mascotas.
- Consulta y gestión de servicios veterinarios.
- Configuración de fechas y horarios disponibles.
- Reserva y cancelación de citas.
- Consulta de citas por parte del cliente.
- Gestión del estado de las citas.
- Diseño adaptable a computadores, tabletas y celulares.
